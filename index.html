2D Farming Sim Set in Space

Pixel Art · Multidisciplinary Team · Unity Game

A 2D farming simulation where players manage their own farm: plant seeds, nurture crops through growth stages, harvest fruits, earn currency, and purchase new seed varieties to expand gameplay progression.

Unity C# Git Multidisciplinary Team
Explore Features View Media
Core loop: Plant → Grow → Harvest → Earn → Shop

Gameplay Features

Plant

Select seeds from toolbar and place them on farmland

Grow

Crops progress through visual growth stages over time

Harvest

Collect mature crops ready for collection

Earn

Harvest generates currency for purchases

Shop

Purchase from 3 seed varieties with varied rewards

Key Highlights

Gameplay Loop

Complete farming mechanics with tools, planting, growth stages, harvesting, and progression via in-game currency.

Progression System

3 seed types with varying growth times and rewards. Initial starter seed plus unlockable premium varieties.

User Interface

Tool toolbar with real-time feedback on selection and currency accumulation for clear player communication.
